Dependencies

oopcorenlp depends on a great many external java libraries. These dependencies are specified in the pom.xml and are managed by maven.

There are two non-java dependencies: wordnet and verbnet. These are available as http downloads and are conventionally expected to be installed one directory above the oopcorenlp installation. See oopcorenlp_ci for more information.

There are a few API key dependencies. While the software will work without them, the results will be better with them.

  • Perfecttense
  • Phrase Finder
  • Azure or
  • Flickr and Faceplusplus
